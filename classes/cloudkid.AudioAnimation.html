cloudkid.AudioAnimation Class

Defined in: src/AudioAnimation.js:12

AudioAnimation Handles playback of a single MovieClip in sync with a sound managed by cloudkid.Audio

Constructor

cloudkid.AudioAnimation

(
  • movieClip
  • soundAlias
  • frameLabel
  • numLoops
  • soundStartFrame
)

Defined in src/AudioAnimation.js:12

Parameters:

  • movieClip createjs.MovieClip

    the animation to sync with sound

  • soundAlias String

    the name of the sound to play with MovieClip

  • frameLabel String

    the alias of the animation sequence to sync with sound Leave blank (or null) to play whole movieClip

  • numLoops Number

    the number of times to play the synced animation when play() is called. value of 0 loops forever. Leave blank to play 1 time (or set as 1)

  • soundStartFrame Number

    frame number on which synced sound starts (EaselJS frame numbers start at "0" where flash is "1")
