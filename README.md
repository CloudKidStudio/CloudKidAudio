CloudKid Audio
=============

HTML5 Audio API for CloudKidOS

##Documentation##

http://cloudkidstudio.github.io/CloudKidAudio/

##Installation##

CloudKid Audio can be install using Bower.

```bash
bower install cloudkid-audio
```
